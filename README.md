# 📊 Sales Analysis Dashboard using Power BI

## 📌 Project Overview
This project is an interactive **Sales Analysis Dashboard** built using **Power BI** to analyze sales performance, profitability, customer behavior, payment preferences, and regional trends.

The dashboard helps uncover key business insights through KPI tracking, category analysis, time-based trends, and geographical performance visualization.

---

# 🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Time Intelligence Functions
- Data Visualization

---

# 📂 Dataset Information
The project uses two datasets:

### 1️⃣ Orders Table
Contains:
- Order ID
- Order Date
- Customer Name
- State
- City

### 2️⃣ Details Table
Contains:
- Order ID
- Category
- Sub-Category
- Amount
- Profit
- Quantity
- Payment Mode

A separate **Calendar Table** was created for time intelligence analysis.

---

# 🔗 Data Model
Relationships used in the project:

- Orders[Order ID] → Details[Order ID] (One-to-Many)
- Calendar[Date] → Orders[Order Date] (One-to-Many)

---

# 📌 Key Insights
- Total Sales reached **438K** with a Total Profit of **37K** across **500 orders** and **336 customers**.
- January recorded the highest monthly sales (**62K**), while July showed the lowest sales performance (**13K**).
- Electronics generated the highest sales contribution (**38%**).
- Clothing generated the highest overall profit contribution (**36%**).
- Maharashtra and Indore emerged as the top-performing state and city respectively.
- Cash on Delivery (COD) was the most preferred payment mode among customers.
